Производство мостовых металлоконструкций

"ЗАВОД ЖБМК производит мостовые металлоконструкции и комплектующие к ним для дальнейшего монтажа металлических мостов различных типов."

Производство арматурных каркасов

"ЗАВОД ЖБМК производит сварные арматурные каркасы для буронабивных свай, колонн, мостовых опор, ригелей и другие объемные арматурные каркасы."

Обработка металла

Завод мостовых конструкций выполняет весь спектр работ по обработке металла. Осуществляем обработку любых видов стали: нержавеющие, легированные, высоколегированные, инструментальные. Производим шпильки из нержавеющей стали для установки шумозащитных экранов.

Перевозка

Мы приветствуем и всегда открыты для новых взаимовыгодных партнёрских отношений. Поэтому если Вам необходимо доставить груз из пункта А в пункт Б, то вы всегда можете обратиться к нашим менеджерам и они с легкостью подберут для вас самые оптимальные условия по перевозке.

Опыт поставок

Мостовые балки и сваи, произведенные ООО «ЗАВОД ЖБМК», использовались при строительстве таких дорожных объектов, как:

  • трасса М12 для ООО «БалтМостСтрой»
  • трасса М3 для АО «МСУ-1»
  • трасса Виноградино-Болтино-Тарасовка для ООО ПКФ «ВиС»
  • трасса М5 для ООО «СК «Кронос»
  • трасса М12 для ООО «А-МОСТ»
